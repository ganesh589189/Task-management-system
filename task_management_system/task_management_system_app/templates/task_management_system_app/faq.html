Task Management System - FAQ

How do I create a new task?

Step 1. Start the "Create Basic Task" wizard. Step 2. Name the task and give it a description. Step 3. Choose when to execute the new task. Step 4. Set the action that you want to be executed. Step 5. Check the details of the new task and save it.

Can I assign a task to multiple users?

You cannot assign a single Jira issue (Task) to multiple users at a time. Best approach is to break the main Task by creating sub-tasks and then assign those sub tasks within the team as needed.

How do I mark a task as complete?

Shift to the Task view by clicking Task in the Navigation Pane. Right clicking on a task you want to mark complete, and then select Mark Complete from the right-clicking menu. And then your selected task has been marked as complete, the item is displayed in gray strikethrough text.
